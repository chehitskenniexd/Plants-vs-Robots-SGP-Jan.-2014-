ReadMe 01 Jan 2015:

Open the PvR.exe in the "Game" folder to run the game.
The save file in the game was built during an Expo to show off the game.
New profiles are not able to be created in the game.

WARNING: BG music can be loud when loading the game.

Notes:
If any code is recomplied, the "SGDWrappers" are done in DX9 and requires DirectX9 to operate.
